# RAGFlow "检索+生成"链路性能优化方案

> 针对Docker容器内部署的优化方案

---

## 一、优化目标

针对RAGFlow "检索+生成"链路的性能瓶颈，引入以下优化机制：

| 优化目标 | 实现方式 |
|---------|---------|
| 异步处理I/O等待 | 异步请求队列 + 信号量并发控制 |
| 流量限流 | 令牌桶算法限流器（租户级别） |
| 防止雪崩 | 熔断器模式保护LLM API |
| 多租户隔离 | 每租户独立限流计数 |

---

## 三、核心模块说明

### 1. 异步请求队列 (`AsyncRequestQueue`)

```python
class AsyncRequestQueue:
    def __init__(self, max_concurrent=10, max_queue_size=1000, max_wait_seconds=300):
        # max_concurrent: 最大并发数
        # max_queue_size: 队列大小
        # max_wait_seconds: 等待超时
```

**功能**：
- 优先级队列：HIGH(0) > NORMAL(1) > LOW(2)
- 信号量控制并发数
- 超时自动拒绝

### 2. 限流器 (`RateLimiter`)

```python
class RateLimiter:
    def __init__(self, requests_per_minute=60, requests_per_second=10):
        # RPM: 每分钟请求数限制
        # RPS: 每秒请求数限制
```

**功能**：
- 令牌桶算法
- 多租户独立计数
- 超出限制自动等待（最多30秒）

### 3. 熔断器 (`CircuitBreaker`)

```python
class CircuitBreaker:
    # 三种状态: CLOSED -> OPEN -> HALF_OPEN -> CLOSED
    def __init__(self, failure_threshold=5, recovery_timeout=30.0):
        # failure_threshold: 失败次数阈值（默认5次）
        # recovery_timeout: 恢复检测间隔（默认30秒）
```

**状态转换**：
```
CLOSED (正常) ──[失败>=阈值]──▶ OPEN (熔断)
    ▲                              │
    │                              │ [等待30秒]
    │                              ▼
    └──[HALF_OPEN成功>=2次]── HALF_OPEN (半开)
```

#
#  RAGFlow Async Queue & Rate Limiter Module
#

import asyncio
import logging
import os
import time
from collections import defaultdict
from enum import IntEnum
import weakref

logger = logging.getLogger(__name__)


class Priority(IntEnum):
    HIGH = 0
    NORMAL = 1
    LOW = 2


class LoopLocalSemaphore:
    """Asyncio semaphore bind to event loop for concurrency control."""

    def __init__(self, value: int):
        self._value = int(value)
        self._semaphores = weakref.WeakKeyDictionary()

    def _get(self):
        loop = asyncio.get_running_loop()
        for cached_loop in list(self._semaphores):
            if cached_loop.is_closed():
                self._semaphores.pop(cached_loop, None)
        sem = self._semaphores.get(loop)
        if sem is None:
            sem = asyncio.Semaphore(self._value)
            self._semaphores[loop] = sem
        return sem

    async def acquire(self):
        return await self._get().acquire()

    def release(self):
        self._get().release()

    async def __aenter__(self):
        await self.acquire()
        return self

    async def __aexit__(self, exc_type, exc, tb):
        self.release()
        return False


class RateLimiter:
    """Token bucket rate limiter with per-tenant limits."""

    def __init__(
        self,
        requests_per_minute: int = 60,
        requests_per_second: int = 10,
    ):
        self._rpm = requests_per_minute
        self._rps = requests_per_second
        self._rpm_buckets: dict[str, float] = defaultdict(lambda: time.time())
        self._rps_buckets: dict[str, float] = defaultdict(lambda: time.time())
        self._rpm_count: dict[str, int] = defaultdict(int)
        self._rps_count: dict[str, int] = defaultdict(int)
        self._lock = asyncio.Lock()

    async def acquire(self, tenant_id: str = "") -> bool:
        now = time.time()
        async with self._lock:
            # Reset RPM counter if 60s passed
            if now - self._rpm_buckets[tenant_id] >= 60:
                self._rpm_buckets[tenant_id] = now
                self._rpm_count[tenant_id] = 0
            # Reset RPS counter if 1s passed
            if now - self._rps_buckets[tenant_id] >= 1:
                self._rps_buckets[tenant_id] = now
                self._rps_count[tenant_id] = 0

            # Check limits
            if self._rpm_count[tenant_id] >= self._rpm:
                logger.warning(f"Tenant {tenant_id} RPM limit reached: {self._rpm_count[tenant_id]}/{self._rpm}")
                return False
            if self._rps_count[tenant_id] >= self._rps:
                logger.warning(f"Tenant {tenant_id} RPS limit reached: {self._rps_count[tenant_id]}/{self._rps}")
                return False

            # Increment counters
            self._rpm_count[tenant_id] += 1
            self._rps_count[tenant_id] += 1
        return True

    async def wait_for_slot(self, tenant_id: str = "", timeout: float = 30.0) -> bool:
        """Wait for a rate limit slot to become available."""
        start_time = time.time()
        while time.time() - start_time < timeout:
            if await self.acquire(tenant_id):
                return True
            await asyncio.sleep(0.1)
        return False


class CircuitBreaker:
    """Circuit breaker pattern for LLM API protection."""

    class State(IntEnum):
        CLOSED = 0
        OPEN = 1
        HALF_OPEN = 2

    def __init__(
        self,
        failure_threshold: int = 5,
        recovery_timeout: float = 30.0,
    ):
        self._failure_threshold = failure_threshold
        self._recovery_timeout = recovery_timeout
        self._state = self.State.CLOSED
        self._failure_count = 0
        self._last_failure_time = 0.0
        self._lock = asyncio.Lock()

    @property
    def state(self):
        return self._state

    async def can_execute(self) -> bool:
        async with self._lock:
            if self._state == self.State.CLOSED:
                return True
            if self._state == self.State.OPEN:
                if time.time() - self._last_failure_time >= self._recovery_timeout:
                    self._state = self.State.HALF_OPEN
                    logger.info("Circuit breaker: OPEN -> HALF_OPEN")
                    return True
                return False
            return True

    async def record_success(self):
        async with self._lock:
            if self._state == self.State.HALF_OPEN:
                self._state = self.State.CLOSED
                self._failure_count = 0
                logger.info("Circuit breaker: HALF_OPEN -> CLOSED")
            elif self._state == self.State.CLOSED:
                self._failure_count = max(0, self._failure_count - 1)

    async def record_failure(self):
        async with self._lock:
            self._failure_count += 1
            self._last_failure_time = time.time()
            if self._state == self.State.HALF_OPEN:
                self._state = self.State.OPEN
                logger.warning("Circuit breaker: HALF_OPEN -> OPEN (failure in half-open)")
            elif self._state == self.State.CLOSED:
                if self._failure_count >= self._failure_threshold:
                    self._state = self.State.OPEN
                    logger.warning(f"Circuit breaker: CLOSED -> OPEN (failures: {self._failure_count})")


# Global instances
CHAT_RATE_LIMITER = RateLimiter(
    requests_per_minute=int(os.environ.get("CHAT_RPM_LIMIT", "60")),
    requests_per_second=int(os.environ.get("CHAT_RPS_LIMIT", "10")),
)

CHAT_CIRCUIT_BREAKER = CircuitBreaker(
    failure_threshold=int(os.environ.get("CIRCUIT_BREAKER_FAILURE_THRESHOLD", "5")),
    recovery_timeout=float(os.environ.get("CIRCUIT_BREAKER_RECOVERY_TIMEOUT", "30.0")),
)
EOF

#
#  RAGFlow Rate Limiter Middleware
#

import asyncio
import logging
import os
from functools import wraps
from typing import Callable

from quart import g, jsonify

from common.async_queue import (
    CHAT_CIRCUIT_BREAKER,
    CHAT_RATE_LIMITER,
)

logger = logging.getLogger(__name__)

RATE_LIMIT_ENABLED = os.environ.get("RATE_LIMIT_ENABLED", "true").lower() == "true"


def get_current_tenant_id() -> str:
    """Get the current tenant ID from the request context."""
    user = getattr(g, "user", None)
    if user and hasattr(user, "tenant_id"):
        return str(user.tenant_id)
    return ""


def rate_limit_chat(func: Callable):
    """
    Decorator for chat completion endpoints with rate limiting and circuit breaker.

    Flow:
    1. CircuitBreaker check -> OPEN: return 503
    2. RateLimiter check -> exceeded: wait 30s or return 429
    3. Execute -> success: record_success / failure: record_failure
    """
    @wraps(func)
    async def wrapper(*args, **kwargs):
        if not RATE_LIMIT_ENABLED:
            return await func(*args, **kwargs)

        tenant_id = get_current_tenant_id()

        # Circuit breaker check
        if not await CHAT_CIRCUIT_BREAKER.can_execute():
            logger.warning(f"Circuit breaker open for chat, tenant {tenant_id}")
            return jsonify({
                "code": 503,
                "message": "Service temporarily unavailable. Please try again later.",
                "data": None,
            }), 503

        # Rate limiter check
        allowed = await CHAT_RATE_LIMITER.acquire(tenant_id)
        if not allowed:
            slot_available = await CHAT_RATE_LIMITER.wait_for_slot(tenant_id, timeout=30.0)
            if not slot_available:
                logger.warning(f"Chat rate limit timeout for tenant {tenant_id}")
                return jsonify({
                    "code": 429,
                    "message": "Too many requests. Please wait a moment and try again.",
                    "data": None,
                }), 429

        try:
            result = await func(*args, **kwargs)
            await CHAT_CIRCUIT_BREAKER.record_success()
            return result
        except Exception as e:
            await CHAT_CIRCUIT_BREAKER.record_failure()
            raise

    return wrapper
EOF

## 七、熔断器状态说明

| 状态 | 含义 | 行为 |
|------|------|------|
| **CLOSED** | 正常 | 所有请求通过，失败计数累积 |
| **OPEN** | 熔断 | 所有请求直接返回503，等待恢复 |
| **HALF_OPEN** | 半开 | 允许少量请求通过测试，成功则关闭 |

**状态转换条件**：
- CLOSED → OPEN：连续失败 >= 5次
- OPEN → HALF_OPEN：等待 >= 30秒
- HALF_OPEN → CLOSED：测试请求成功
- HALF_OPEN → OPEN：测试请求失败

## 八、故障排除

### 问题1：模块导入失败

```
ModuleNotFoundError: No module named 'api.middleware'
```

**解决**：
```bash
# 确保创建了 __init__.py

```

### 问题2：限流不生效

**解决**：
```bash
# 检查环境变量
docker exec docker-ragflow-cpu-1 env | grep RATE_LIMIT

# 确认已设置为 true
```

### 问题3：503 Service Unavailable

**原因**：熔断器已打开

**解决**：
```bash
# 查看熔断器状态
docker exec docker-ragflow-cpu-1 python3 -c "from common.async_queue import CHAT_CIRCUIT_BREAKER; print(CHAT_CIRCUIT_BREAKER.state)"

# 等待30秒自动恢复，或重启容器
docker restart docker-ragflow-cpu-1
```

### 问题4：429 Too Many Requests

**原因**：触发了限流

**解决**：
- 等待片刻后重试
- 或调整环境变量 `CHAT_RPM_LIMIT` 和 `CHAT_RPS_LIMIT`
